# Avatar Image

Place your avatar image here as `avatar.jpg`

The homepage expects:
- **Path**: `/public/images/avatar.jpg`
- **Format**: JPG (preferred) or PNG
- **Recommended size**: 400x400px or larger (will be displayed at 128-160px)
- **Aspect ratio**: Square (1:1)

The image will be:
- Displayed as a circle with a border ring
- Responsive: 128px on mobile, 160px on desktop
- Optimized by Next.js Image component
- Loaded with priority (above-the-fold)
