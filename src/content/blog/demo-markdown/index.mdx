## Text Formatting

You can use **bold text** for emphasis, _italic text_ for subtle emphasis, and **_bold italic_** for maximum emphasis. Use ~~strikethrough~~ for deleted text.

Inline `code snippets` are styled with <GlossaryTooltip term="Monospace" definition="A font family where every character occupies the same horizontal space, commonly used for code display to maintain alignment and distinguish code from prose">monospace</GlossaryTooltip> font and background highlighting. You can also use <kbd>Cmd</kbd> + <kbd>K</kbd> to show keyboard shortcuts.

This paragraph includes a [link to Next.js](https://nextjs.org/) and an external link with proper styling.

<KeyTakeaway variant="tip" title="Typography Hierarchy">
  The blog uses a responsive typography scale with six heading levels. Lora serif provides optimal
  screen reading for body text, while Geist Sans handles interface labels and navigation. Both fonts
  support dark mode through CSS variables.
</KeyTakeaway>

## Math & LaTeX (KaTeX)

Inline math equations render with <GlossaryTooltip term="KaTeX" definition="Fast math typesetting library for the web, based on TeX. Renders LaTeX equations at build time without client-side JavaScript">KaTeX</GlossaryTooltip>: $E = mc^2$ is Einstein's famous equation.

Block math equations are centered on their own line:

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

Complex expressions work too:

$$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
